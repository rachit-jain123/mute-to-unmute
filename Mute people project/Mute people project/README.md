# Mute to Unmute 🔇➡️🔊

A web-based accessibility platform designed to help people with speaking disabilities express themselves freely — without speech.

## 🌐 Live Features

| Feature | Description |
|--------|-------------|
| 🗣️ Text Reader | Converts text to speech using Web Speech API |
| 🎵 Music Player | Spotify-style music player for relaxation |
| 🎨 Paint Tool | Canvas-based drawing tool to express thoughts visually |
| 😊 General Feelings | Image-based emotion communication board |
| 🏥 Doctor Appointment | Book appointments with ease |
| 🤝 Donor & Receiver | Connect people who want to help |

## 🛠️ Tech Stack

- **HTML5** — Structure & Semantics
- **CSS3** — Responsive Design, Flexbox
- **JavaScript** — DOM Manipulation, Events
- **Web Speech API** — Text to Speech
- **Canvas API** — Paint Tool
- **Audio API** — Music Player

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/rachit-jain123/mute-to-unmute.git
```
2. Open `index.html` in any browser
3. No server setup required for frontend features
