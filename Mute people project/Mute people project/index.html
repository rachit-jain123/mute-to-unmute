about the project

What makes this project special?

Mute to Unmute is built for people with speaking disabilities. It provides tools like a Text Reader (Web Speech API), Music Player, Paint tool, and a General Feelings board — enabling communication without speech.

The project also includes a Login system, Doctor Appointment booking, and a Donor-Receiver management portal — making it a complete accessibility solution.
